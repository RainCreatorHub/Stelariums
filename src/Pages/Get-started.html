Comece Agora

1

Crie sua conta

Registre-se gratuitamente em nossa plataforma. O processo leva menos de 30 segundos e não requer cartão de crédito.

curl -X POST https://api.site-test.com/v1/register \ -H "Content-Type: application/json" \ -d '{"email": "seu@email.com", "password": "sua-senha-segura"}'
2

Instale o SDK

Adicione nossa biblioteca ao seu projeto usando npm, yarn ou CDN. Nós suportamos todas as principais frameworks modernos.

// npm npm install site-test-sdk // yarn yarn add site-test-sdk // CDN
3

Configure sua primeira integração

Use nosso assistente de configuração para conectar sua aplicação. Basta seguir os passos no painel administrativo.

// Inicialização básica const client = new SiteTestClient({ apiKey: 'sua-chave-api-aqui', environment: 'production' }); // Iniciar serviço client.start().then(() => { console.log('Site-Test iniciado com sucesso!'); });
